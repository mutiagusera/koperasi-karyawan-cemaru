Semua data di atas tersimpan lokal di browser ini (localStorage), tanpa internet. Lakukan backup secara berkala agar data tidak hilang jika cache browser dibersihkan atau ganti device.

⬇️ Backup (Unduh Data)

Backup JSON dipakai untuk memulihkan data nanti (cadangan lengkap & akurat). Backup Excel dipakai untuk dilihat, dicetak, atau diolah di Excel/Spreadsheet — tidak bisa di-import kembali.

⬆️ Restore (Pulihkan Data)

Pilih file backup JSON (.json) yang pernah diunduh. Perhatian: proses ini akan menggantikan seluruh data yang ada saat ini di aplikasi dengan data dari file backup.
🏪 Koperasi PT. Cemaru Lestari — backup rutin, koperasi lebih aman

🗑️ Sampah (Undo Hapus)

Berisi maksimal 30 item terbaru yang baru saja dihapus (barang, transaksi, pembayaran, riwayat stok). Item bisa dipulihkan selama masih ada di daftar ini. Item lama akan otomatis terdesak keluar oleh item baru yang dihapus.
